# 实验2：搭建 VPN 服务器

## 实验目的

使用 OpenVPN 搭建 VPN 服务器，实现客户端通过隧道安全访问内网资源。

## 实验要求

1. 安装 OpenVPN 和 easy-rsa
2. 生成 CA 证书、服务器/客户端证书
3. 配置 OpenVPN 服务端
4. 启动并验证服务

## 前置条件

- 两台 Rocky Linux 9 虚拟机（一台服务器、一台客户端）
- 服务器 IP 地址已知

## 操作步骤

### 1. 在服务器上执行

```bash
sudo bash setup.sh
```

### 2. 在客户端上执行

将服务端生成的客户端证书拷贝到客户端，然后执行客户端配置（脚本`setup.sh`默认只配服务端）。

### 3. 验证配置

```bash
sudo bash verify.sh
```

## 验收标准

- OpenVPN 服务正常运行
- tun0 网卡已创建
- 服务器 1194/udp 端口在监听
